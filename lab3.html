Installation de Docker sur Ubuntu

  • Docker est un projet open-source sous licence Apache 2 pour automatiser le dploiement d'applications sous forme de conteneurs.

  • Il ya deux mthodes pour installer Docker sur Ubuntu :

    • Mthode 1 : installer Docker partir du dpt officiel Docker
    • Mthode 2 : installer Docker partir du dpt officiel de votre distribution Ubuntu

  • Pour installer Docker, vous devez choisir soit mthode 1, soit mthode2.

Accord des privilges ncessaires

  • Pour qu'un utilisateur puisse se servir de la commande "docker", il doit appartenir au groupe "docker".

  • Le groupe "docker" accorde des privilges quivalents l'utilisateur root

  • Ainsi, pour viter davoir utiliser sudo lorsque vous utilisez le docker, vous devez appartenir au groupe "docker"

      - Si ce groupe n'existe pas encore sur votre ordinateur, vous devez le crer avec la commande "groupeadd".
In [ ]:
$sudo groupadd docker

-  Ajoutez ensuite l'utilisateur au groupe, avec la commande "usermod".
In [ ]:
$sudo usermod -aG docker $USER
  • L'option "-aG" permet d'ajouter un utilisateur au groupe qui suit. La variable d'environnement "$USER" contient l'utilisateur en cours.
  • Redmarrez pour que les modifications
